Frequently Asked Questions

How is the JSON structured?

Each row becomes a JSON object where column headers become keys and cell values become values. The result is an array of objects.

Can I use this JSON in MongoDB?

Yes! The JSON array format is perfect for MongoDB insertMany() operations or importing via MongoDB Compass.

What about empty cells?

Empty cells are converted to empty strings or null values in JSON, maintaining data structure integrity.

Does it preserve data types?

Numbers stay as numbers and text as strings. Dates are converted to ISO format for maximum compatibility.
