Frequently Asked Questions

What JSON formats are supported?

We support JSON arrays (ideal for tables), single objects, and nested structures. The converter auto-detects and handles all formats.

How are nested objects converted?

Nested objects are flattened with dot notation (e.g., "user.name", "address.city") to create separate columns.

Can I convert Postman/API response data?

Yes! Simply copy your API response JSON and paste it directly. Works great with Postman, Insomnia, or browser DevTools responses.

What's the maximum JSON size?

Since processing is browser-based, there's no server limit. Large files (10MB+) work but may take longer depending on your device.
